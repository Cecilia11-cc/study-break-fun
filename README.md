# 🎯 复习间隙小游戏合集 | Study Break Fun Pack

> 考试复习累了？来个 5 分钟的大脑切换！三个精选互动项目，激活大脑、释放创意、放松解压。

---

## 📦 包含项目

### 🧠 [脑力激活小游戏](脑力激活小游戏.html)
六个快速脑力挑战，给大脑换一个节奏：
- **🃏 记忆翻牌** — 8 对 emoji 卡片，记住位置、配对消除
- **🔢 心算挑战** — 20 道限时口算，越做越难，刷连击
- **🌈 色词冲突** — 经典 Stroop 效应，念颜色不念字
- **⚡ 反应点击** — 测测你的反应速度，挑战 200ms 闪电反应
- **💣 扫雷** — 经典扫雷，初级/中级/高级三种难度
- **🕷️ 蜘蛛纸牌** — 单花色蜘蛛纸牌，K→A 顺子消除
- **🐍 贪吃蛇** — 经典街机，方向键/WASD/触屏滑动操控
- **📐 高数挑战** — 极限、积分、微分方程、级数、几何五大模块题库
- **📚 汉语言知识** — 现代汉语、古代汉语、文学常识、修辞、文化、诗词按考纲分类
- **💻 编程挑战** — Python/JS/SQL/C 代码填空，15关闯关

### 🎨 [创意画室](创意画室.html)
六个创意绘画项目，放松右脑：
- **✏️ 自由涂鸦** — 多色画笔，随心所欲
- **🪞 对称万花筒** — 画一笔，多方向镜像对称
- **🔲 像素画板** — 格子填色，做像素小图标
- **🌸 曼陀罗绘制** — 极坐标对称花纹
- **🔗 连点成画** — 按数字连线揭晓隐藏图案
- **🌫️ 喷漆涂鸦** — 喷枪效果，街头风格

### 🧸 [解压实验室](解压实验室.html)
四个解压互动玩具，纯粹放松：
- **🫧 捏泡泡纸** — 啪！啪！啪！根本停不下来
- **🎆 粒子烟花** — 点击释放绚丽粒子烟花
- **🌈 彩虹流体** — 流动粒子跟随指尖
- **🎹 迷你钢琴** — 7种乐器音色，键盘/触屏弹奏
- **🌷 花卉种植** 🚧 — 100种花卉，播种浇水施肥，生长模拟（开发中）

### 🧱 [像素方块世界](voxel-world/)
3D 体素沙盒游戏，自由建造与探索：
- **🌍 程序化地形** — 自动生成草地、树木、水域
- **🧱 6 种方块** — 草地、泥土、石头、木头、树叶、水
- **🐄 动物 AI** — 场景中有自由活动的小动物
- **🎮 第一人称** — WASD 移动，鼠标视角，左右键放置/破坏
- **📱 支持移动端** — 虚拟摇杆 + 触控按钮

## 🚀 使用方式

1. 下载/克隆整个文件夹
2. 直接用浏览器打开任意 `.html` 文件
3. 无需安装任何东西，无需网络

```
study-break-fun/
├── README.md
├── LICENSE
├── index.html
├── 脑力激活小游戏.html
├── 创意画室.html
├── 解压实验室.html
└── voxel-world/
    ├── index.html
    ├── styles/main.css
    └── js/
        ├── game.js
        ├── voxel.js
        ├── animals.js
        └── noise.js
```

## ✨ 特点

- 🎨 精美的深色主题 UI + 流畅动画
