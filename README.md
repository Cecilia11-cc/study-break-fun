# 🎯 复习间隙小游戏合集 | Study Break Fun Pack

> 考试复习累了？来个 5 分钟的大脑切换！三个精选互动项目，激活大脑、释放创意、放松解压。

---

## 📦 包含项目

### 🧠 [脑力激活小游戏](脑力激活小游戏.html)
四个快速脑力挑战，给大脑换一个节奏：
- **🃏 记忆翻牌** — 8 对 emoji 卡片，记住位置、配对消除
- **🔢 心算挑战** — 20 道限时口算，越做越难，刷连击
- **🌈 色词冲突** — 经典 Stroop 效应，念颜色不念字
- **⚡ 反应点击** — 测测你的反应速度，挑战 200ms 闪电反应

### 🎨 [创意画室](创意画室.html)
六个创意绘画项目，放松右脑：
- **✏️ 自由涂鸦** — 多色画笔，随心所欲
- **🪞 对称万花筒** — 画一笔，多方向镜像对称
- **🔲 像素画板** — 格子填色，做像素小图标
- **🌸 曼陀罗绘制** — 极坐标对称花纹
- **🔗 连点成画** — 按数字连线揭晓隐藏图案
- **🌫️ 喷漆涂鸦** — 喷枪效果，街头风格

### 🧸 [解压实验室](解压实验室.html)
四个解压互动玩具，纯粹放松：
- **🫧 捏泡泡纸** — 啪！啪！啪！根本停不下来
- **🎆 粒子烟花** — 点击释放绚丽粒子烟花
- **🌈 彩虹流体** — 流动粒子跟随指尖
- **🎹 迷你钢琴** — 弹几个音符，还能用键盘弹

## 🚀 使用方式

1. 下载/克隆整个文件夹
2. 直接用浏览器打开任意 `.html` 文件
3. 无需安装任何东西，无需网络

```
study-break-fun/
├── README.md
├── LICENSE
├── 脑力激活小游戏.html
├── 创意画室.html
└── 解压实验室.html
```

---

## ✨ 特点

- 🎨 精美的深色主题 UI + 流畅动画
- 🔊 内置音效反馈（Web Audio API）
- 📱 支持鼠标 + 触屏（电脑/平板/手机都能玩）
- 💾 画作可保存为 PNG 图片
- 🏆 游戏最高分自动记录（浏览器 localStorage）
- 🚫 零依赖，纯 HTML/CSS/JS 单文件
