高等数学CⅡ

7/6 08:30-10:30 · 2303 · 屈力进 · 还剩 4 天

一、定积分（计算+应用，必考大题）

  • 牛顿-莱布尼茨公式：先求原函数，再代入上下限
  • 换元积分法 —— 换元必换限！
  • 分部积分：∫udv = uv − ∫vdu
  • 变上限积分求导：d/dx ∫aφ(x) f(t)dt = f[φ(x)]·φ'(x)
  • 应用：旋转体体积 V = π∫[f(x)]²dx、平面面积
  • 反常积分：无穷限/瑕积分，先求积分后取极限

二、微分方程（一阶+二阶，计算题核心）

  • 可分离变量：dy/dx = g(x)h(y) → 分离积分
  • 一阶线性：y' + P(x)y = Q(x)，通解公式必须背
  • 二阶常系数齐次：特征方程 r²+pr+q=0，三种根对应三种通解
  • 二阶非齐次：通解 = 齐次通解 + 特解（待定系数法）

三、无穷级数（判敛+展开，区分度高）

  • 几何级数 ∑qⁿ（|q|<1 收敛）、p-级数 ∑1/nᵖ（p>1 收敛）
  • 正项级数：比值判别法（达朗贝尔）、比较判别法
  • 交错级数：莱布尼茨判别法
  • 幂级数收敛半径 R = lim|aₙ/aₙ₊₁|，端点单独判
  • 麦克劳林展开：eˣ, sinx, cosx, ln(1+x), (1+x)ᵅ 必须背

复习策略

数学靠刷题，不能只看不写。每天至少 2h 动手算。错题反复过。先做近 2-3 年真题了解命题风格。

微观经济学A

7/10 08:30-10:30 · 2301 · 刘迪 · 还剩 8 天

一、供需与弹性（20-25%，最高频考点）

  • 需求价格弹性 Ed = (ΔQ/Q) / (ΔP/P)
  • Ed>1 富有弹性 → 薄利多销（降价增收）
  • Ed<1 缺乏弹性 → 谷贱伤农（丰收减收）
  • 收入弹性 >0 正常品，<0 低档品
  • 交叉弹性 >0 替代品，<0 互补品

二、边际分析（贯穿全课程的方法论）

  • 消费者均衡：MU₁/P₁ = MU₂/P₂ = λ
  • 生产者均衡：MPL/w = MPK/r
  • 利润最大化：MR = MC（全市场通用）
  • MC 过 AC 最低点（收支相抵点），MC 过 AVC 最低点（停止营业点）

三、四种市场结构（论述题最爱）

市场厂商产品长期利润效率
完全竞争极多同质0最高
垄断竞争较多有差异0有剩余产能
寡头少数->0较低
完全垄断1个无替代超额利润最低(无谓损失)

四、其他高频

  • 市场失灵：垄断、外部性、公共物品、信息不对称
  • 帕累托最优：MRSA=MRSB（交换），MRTSL=MRTSK（生产）
  • 吉芬商品：收入效应 > 替代效应
  • 价格歧视三级公式

复习策略

画图是核心 —— 供需均衡图、成本曲线族、四种市场均衡图必须手到擒来。每天画一遍核心图表。

🥗 考前饮食建议

⚠️ 考试当天饮食原则

早餐：必须吃！蛋白质+碳水（鸡蛋+全麦面包+牛奶），不要只吃甜的。
考前 1h：一小块黑巧克力或香蕉，稳定血糖。
考场带水：温水，小口喝。别喝冰的。
避免：油炸、过辣、过甜、碳酸饮料——容易犯困或胃不适。
咖啡：平时不喝的话考前别尝试，心跳加速反而影响发挥。
